### Prerequisites

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Fire7610/Studio-402-Website.git
cd Studio-402-Website
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

5. Preview production build:
```bash
npm run preview
```
